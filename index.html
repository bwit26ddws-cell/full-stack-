The Full Stack Landscape

Front-End (Client-Side)

What runs in the user's browser — HTML, CSS, JavaScript, and frameworks like React, Vue, or Angular.

HTML — Structure

HTML (HyperText Markup Language) is the foundation of all web pages. It provides the structure and semantic meaning to content using a system of tags and elements.

What is HTML?

HTML is a markup language, not a programming language. It uses tags (marked with angle brackets) to tell browsers how to structure and display content. Every HTML document starts with a doctype declaration and contains a head and body section.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

Common HTML Elements

  • <h1> to <h6> — Headings (h1 is the most important)
  • <p> — Paragraphs of text
  • <ul>, <ol>, <li> — Lists (unordered, ordered, items)
  • <a> — Links to other pages or resources
  • <img> — Embed images
  • <form>, <input>, <textarea> — Forms for user input
  • <div>, <section>, <article> — Container elements
  • <header>, <footer>, <nav> — Semantic layout elements

CSS — Presentation

CSS (Cascading Style Sheets) is responsible for the visual presentation of your HTML. It controls colors, layouts, spacing, fonts, animations, and responsive design.

What is CSS?

CSS uses selectors to target HTML elements and applies rules (properties and values) to style them. Modern CSS includes powerful features like custom properties (variables), flexbox, CSS Grid, and media queries for responsive design.

CSS Syntax

selector {
  property: value;
  another-property: value;
}

Example

h1 {
  color: #4F8EFF;
  font-size: 2.5rem;
  text-align: center;
}

CSS Custom Properties (Variables)

Custom properties allow you to store reusable values and update them dynamically:

:root {
  --accent-color: #4F8EFF;
  --bg-dark: #0D0D1A;
}

p {
  color: var(--accent-color);
}

Modern CSS Features

Flexbox

Layout items in a row or column with powerful alignment and distribution options.

CSS Grid

Create complex 2D layouts with rows and columns, perfect for page designs.

Media Queries

Apply different styles based on screen size for responsive, mobile-friendly design.

Animations

Create smooth transitions and animations to enhance user experience.

Key Takeaways

  • Use CSS custom properties for consistent, maintainable styling
  • Flexbox and Grid are powerful layout tools — master both
  • Mobile-first approach: design for mobile, then scale up
  • Use media queries to create responsive designs
  • Keep CSS organized and use meaningful class names

Bootstrap — Framework

Bootstrap is the world's most popular CSS framework. It provides a comprehensive set of pre-built components, a powerful grid system, and utilities for rapid, responsive web development.

What is Bootstrap?

Bootstrap is a collection of CSS and JavaScript files that provide:

  • A 12-column responsive grid system
  • Pre-styled components (buttons, cards, forms, modals)
  • Utility classes for spacing, sizing, and display
  • JavaScript plugins for interactive components
  • Consistent, professional design out of the box

Bootstrap Grid System

<div class="container">
  <div class="row">
    <div class="col-md-6">Half width</div>
    <div class="col-md-6">Half width</div>
  </div>
</div>

Bootstrap Components

Bootstrap comes with many ready-to-use components:

Buttons

.btn .btn-primary .btn-secondary etc. Multiple styles and sizes.

Cards

.card component for content containers with headers, bodies, and footers.

Forms

Styled form controls, input groups, and validation messages.

Navbar

Responsive navigation bar that collapses on mobile.

Modals

Dialog boxes for alerts, confirmations, and user input.

Carousel

Image slider with controls for showcasing multiple items.

Bootstrap Utilities

Bootstrap's utility classes let you apply common styles without writing CSS:

<div class="mt-4 mb-2 p-3 bg-light text-center">
  Margin top, margin bottom, padding, background, text alignment
</div>

Responsive Design

Bootstrap's grid automatically adapts to different screen sizes, ensuring your site looks great on mobile, tablet, and desktop.

Pre-built Components

Customization

  • Bootstrap provides a 12-column grid system — use it for responsive layouts

  • Bootstrap components save development time — leverage them

JavaScript — Behavior

JavaScript is the programming language of the web. It runs in the browser and adds interactivity, responds to user events, manipulates the DOM, and communicates with servers.

What is JavaScript?

JavaScript is an event-driven, dynamic programming language that allows you to:

  • Respond to user interactions (clicks, typing, scrolling)
  • Manipulate the DOM — add, remove, or modify HTML elements
  • Update content without reloading the page (AJAX)
  • Validate form input before sending to the server
  • Create animations and transitions

JavaScript Basics

// Variables and data types
let name = "Diwas";
let age = 20;
const isActive = true;

// Functions
function greet(name) {
  return "Hello, " + name;
}

// Event listeners
button.addEventListener("click", function() {
  console.log("Button clicked!");
});

DOM Manipulation

The DOM (Document Object Model) is a JavaScript representation of the HTML page. You can query and modify it:

// Select elements
let element = document.getElementById("myId");
let items = document.querySelectorAll(".item");

// Modify content
element.textContent = "New text";
element.style.color = "blue";

// Add/remove classes
element.classList.add("active");
element.classList.remove("hidden");

JavaScript Concepts

Events

User interactions like click, keyup, submit, mouseover trigger event handlers.

Functions

Reusable blocks of code that perform specific tasks. Can accept parameters and return values.

Arrays & Objects

Collections of data that you can iterate over and manipulate.

Async/Await

Handle asynchronous operations like fetching data from APIs.

Fetching Data from APIs

async function fetchData() {
  try {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error("Error:", error);
  }
}

Key Takeaways

  • Use event listeners to respond to user interactions
  • Manipulate the DOM to update page content dynamically
  • Write reusable functions to keep code organized
  • Use async/await for cleaner asynchronous code
  • Always validate user input before processing
